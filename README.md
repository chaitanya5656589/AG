# HealthTrack App

A cross-platform hybrid health records tracking mobile application. This is a high-fidelity web mockup designed as a Progressive Web App (PWA).

## Features
- **Mobile-First Design**: Built with a responsive mobile frame.
- **Login w/ OTP**: Simulated secure authentication flow.
- **Dashboard**: Quick access to Hospitals, Reports, Diseases, and Medical Bills.
- **Reports Management**: Add new reports via simulated camera capture.
- **Document Scanner**: Built-in camera overlay for document scanning.

## Tech Stack
- **HTML5**: Semantic structure.
- **TailwindCSS**: Utility-first CSS framework (via CDN).
- **Vanilla JS**: Lightweight, framework-agnostic logic.
- **FontAwesome**: Icon library.

## How to Run
1.  Open `index.html` in any modern web browser.
2.  (Optional) Serve with `live-server` or `http-server` for better performance.

## Deployment (Android)
To deploy this as an Android App:
1.  Install Node.js.
2.  Initialize a Capacitor project: `npm install @capacitor/core @capacitor/cli`.
3.  Add Android platform: `npx cap add android`.
4.  Build and sync: `npx cap sync`.
5.  Open in Android Studio: `npx cap open android`.
